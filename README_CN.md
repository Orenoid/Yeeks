# [Yeeks](https://yeeks.orenoid.com/)

一个以周为单位展示全年时间的可视化工具。

## 使用方法

1. 安装依赖：
```bash
npm install
# 或
yarn
```

2. 启动开发服务器：
```bash
npm run dev
# 或
yarn dev
```

3. 构建生产版本：
```bash
npm run build
# 或
yarn build
```

## 灵感来源

本项目灵感来源于 [Your Life in Weeks](https://waitbutwhy.com/2014/05/life-weeks.html)
