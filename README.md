# Yeeks


Your year in weeks.

## Getting Started

1. Install dependencies:
```bash
npm install
# or
yarn
```

2. Start the development server:
```bash
npm run dev
# or
yarn dev
```

3. Build for production:
```bash
npm run build
# or
yarn build
```
